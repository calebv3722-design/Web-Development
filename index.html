The Problem and Solution with Cookie and Privacy Applications

This project explores modern web technologies with a focus on how cookies, privacy controls, and secure browser practices shape user experience and data protection on the web. These topics include how cookies store user information and how websites can track users input throughout different websites with the consistent threats of third-party cookies. The solutions we provide are Secureflag, HttpOnly, and SameSite attributes to help fix the secruity risk cookies and privary controls provide. Additionally, in later sections we analyze broader technologies and their role in today's rapidly evolving web ecosystem.
